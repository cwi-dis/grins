GRiNS Player for Macintosh Release Notes

These are the release notes for the GRiNS player for Macintosh, Release 0.2, patchlevel 0. More information on GRiNS can be obtained at http://www.cwi.nl/GRiNS, or by email to grins-request@cwi.nl.

What (sparse) documentation on GRiNS is available at this time can also be found on the homepage.

Prerequisites

You need a PPC or 68K macintosh with a minimum of 16Mb of RAM and System 7.1. For decent multimedia performance, however, you will probably need at least a 100Mhz PPC with 32Mb of RAM. You may want to play with the application size through the "File info" window in the Finder.

You need MacTCP or Open Transport to access documents over the net (included in MacOS since System 7.5).

GRiNS uses QuickTime and Sound Manager for multimedia playback. You need QuickTime 2.5 or later. QuickTime 3 includes MPEG support, for QuickTime 2.5 you have to get a separate MPEG extension. QuickTime and the extension can be obtained from quicktime.apple.com.

Installation instructions

There are none, really: unpacking the StuffIt archive is all you need to do. You may want to tell your text editor to save documents with creator GRIN, type TEXT, to open them automatically with the player when you double-click them.

Customization

One aspect of GRiNS can be customized: the handling of the test attributes such as bitrates and language (see section 4.4 of the SMIL specification). See the examples section on the homepage for details.

Feedback

Bug reports and other technical feedback should be sent to grins-support@cwi.nl, please check the feedback section on the homepage for details.

Non-technical feedback should be sent to grins-request@cwi.nl.

The SMIL language

The GRiNS player supports the complete SMIL 1.0 definition except where noted below. A conforming SMIL document can be correctly parsed, but a few features are not implemented.

Platform dependencies in media handling

There are some differences between the various platforms. See the chart of supported media types on the GRiNS website for the supported media types on the various supported platforms.

Access methods

The GRiNS player supports the following access methods (URL schemes).

  • file
  • http
  • ftp
  • data
  • gopher

Unsupported SMIL features

There are only a few SMIL features that are not supported by the GRiNS player. Documents that make use of these features are parsed correctly, but the features are ignored.

  • begin and end attributes in the anchor element.
  • fit="fill" and fit="scroll" attribute values in region element.
  • name="pics-label" attribute value in meta element.
  • alt attribute in media object elements.
  • fill attribute.

Platform-specific issues

There are also a number of features that are not supported on all platforms, or not supported identically. The following problems are known for the player on the Macintosh:

  • Clip-begin, clip-end and looping are not supported on video items.
  • Double-clicking a document while the player is already running does not work. Open the file from the player File menu in stead.
  • HTML support is very weak, only a few of the major tags are supported, and not even fully.

As more issues are reported they are added to the release notes section on the home page.
