GRiNS Editor for SMIL Release 1.0 Release Notes for Macintosh

These are the release notes for the GRiNS Editor and Player for SMIL Release 1.0 for Macintosh. These release notes were last updated on August 27 1999. For a more up-to-date version of these notes, look here.

Mac user interface open issues

Depending on the settings in the MacOS General Preferences control panel the New command will sometimes not display the contents of the Templates folder but of some other random folder. Navigating to the Templates folder by hand should fix this.

Attempting to play back RealMedia datatypes that are not supported by the version of the G2 player that is installed on your system may result in a crash.

Popup menus near the bottom of the screen tend to come up mostly empty. They do scroll, however, so all menu entries are selectable, it is merely not very pretty.

The HTML renderer in GRiNS for the Macintosh is very limited.

Occasionally there may be problems with popup menus showing the wrong content if you have products installed that add global menus to the menubar, such as Palm Desktop. If this happens clicking a different menu once will make the problem go away.

Under low-memory conditions you may receive incorrect error messages when playing media items. Serious low-memory conditions may lead to a crash.

Access Methods

The GRiNS Editor and Player for SMIL supports the following access methods (URL schemes):

  • file
  • http
  • ftp
  • data
  • gopher

It is possible to create and edit documents that make use of other access methods (in particular the rtsp access method), but they cannot be previewed.

Unsupported SMIL Features

There are only a few SMIL features that are not supported by the GRiNS Editor and Player for SMIL. Documents that make use of these features are parsed correctly, but the features are ignored.

  • begin and end attributes in the anchor element.
  • fit="fill" and fit="scroll" attribute values in region element.
  • name="pics-label" attribute value in meta element.
  • alt attribute in media object elements.
  • fill attribute.
