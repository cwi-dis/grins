GRiNS for RealSystem G2, version 1.5beta 4 Release Notes for Macintosh

These are the release notes for GRiNS for RealSystem G2, version 1.5beta for Macintosh. This file contains release notes for both the Light and the Pro version. For a more up-to-date version of these notes, see http://www.oratrix.com/GRiNS.

New features in this release

Conversion of QuickTime and MPEG movies to RealVideo now works, although only the video track is converted, not the audio track.

The previous beta release had serious memory leaks leading to crashes. These have all been fixed (to our knowledge: please report to us if there are still problems).

Lots of minor bug fixes, interface enhancements, etc.

Open issues

PNG image files are supported in this release, but some PNG images have trouble with the colors.

There appear to be occasional problems when using GRiNS with RealPlayer 7 that we are investigating. Please send us feedback on this issue, at grins-support@oratrix.com. Often when preview play does not work trying it a second time will work. These problems are specific to RealPlayer 7, RealPlayer G2 (also known as RealPlayer 6) works fine.

RealNetworks suggest having at least 32Mb of real memory and 64Mb of virtual memory. We strongly suggest you follow these guidelines, as low-memory conditions may otherwise result in rather spectacular crashes. You should also have RealPlayer 6.0.6.17 or later installed.

Occasionally there may be problems with popup menus showing the wrong content if you have products installed that add global menus to the menubar, such as Palm Desktop. If this happens clicking a different menu once will make the problem go away.
