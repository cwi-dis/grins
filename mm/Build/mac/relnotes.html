GRiNS for Macintosh Release Notes

These are the release notes for the GRiNS editor and player for the Macintosh, Release 0.5, patchlevel 0. More information on GRiNS can be obtained at http://www.oratrix.com/GRiNS, or by email to grins-request@oratrix.com.

What (sparse) documentation on GRiNS is available at this time can also be found on the homepage.

Prerequisites

For decent performance you probably need at least a 100Mhz PowerPC with 32Mb, but even on a 68020 mac with 16Mb of virtual memory things will work (albeit slowly). You need about 15Mb of free diskspace. You need MacOS 7.5 (but 7.1 may work) and QuickTime 2.5 (QuickTime 3 if you want MPEG video playback).

Feedback

Bug reports and other technical feedback should be sent to grins-support@oratrix.com, please check the feedback section on the homepage for details.

Non-technical feedback should be sent to info@oratrix.com.

Platform dependencies in media handling

There are some differences between the various platforms. See the chart of supported media types on the GRiNS website for the supported media types on the various supported platforms.

Access methods

The GRiNS player supports the following access methods (URL schemes).

  • file
  • http
  • ftp
  • data
  • gopher

Unsupported SMIL features

There are only a few SMIL features that are not supported by the GRiNS player. Documents that make use of these features are parsed correctly, but the features are ignored.

  • begin and end attributes in the anchor element.
  • fit="fill" and fit="scroll" attribute values in region element.
  • name="pics-label" attribute value in meta element.
  • alt attribute in media object elements.
  • fill attribute.

Mac user interface open issues

Depending on the settings in the MacOS General Preferences control panel the New command will sometimes not display the contents of the Templates folder but of some other random folder. Navigating to the Templates folder by hand should fix this.

Popup menus near the bottom of the screen tend to come up mostly empty. They do scroll, however, so all menu entries are selectable, it is merely not very pretty.

HTML problems

The HTML renderer in GRiNS for the Macintosh is very limited.

Other open issues issues

There are also a number of features that are not supported on all platforms, or not supported identically.

As issues are reported they are added to the release notes and open issues sections on the home page.
