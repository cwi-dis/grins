File Menu

Using the commands in this menu will enable opening and closing of documents for playback in the Player. Using the Preferences... menu option you can personalize your Player options for playing all documents.

Open...

Using this command you can open files which reside on your hard drive or local network. Selecting a file with the extension .smi, .smil, or .grins will be opened in the player ready for playing.

See also: Opening files

 

Open URL...

This allows you to open a file which is held locally on your hard drive or network. You can also copy a URL from a other document and paste the location here which will then be loaded into the player ready for playing.

See also: Opening files

 

Open recent

The last 5 files opened in the player are listed in this sub menu. Selecting a file from this menu will reopen the file.

See also: Opening files

 

Close document

This closes the currently selected document.

 

Preferences...

This dialogue enables you to set the values used internally by the Player whenrendering SMIL documents:

System bitrate: Choose one of the following settings:

14.4K modem, 19.2K Connection, 28.8K Modem, 33.6K Modem, 56K Modem, 56K Single ISDN, 112K Dual ISDN, 256Kbps DSL/Cable, 300Kbps DSL/Cable, 512Kbps DSL/Cable, T1/LAN, 10Mbps LAN

System language: Choose the preferred language for viewing information. The default is set to 'English'.

Subtitles preferred over overdub :Turning this option on will allow you to view overdub information which may be included in the document to be played.

Show closed-captions when available: Checking this option will permit the viewing of captions which may be part of the document.

Use audio descriptions when available: Setting this option to on will enable you to view descriptions of audio which may be incorporated in the document.

Use Internet Explorer HTML control: Selecting this option will change the control of HTML information to Internet Explorer, (if installed on your system), instead of the default HTML-2 renderer which is included with GRiNS Player.

Dialogue buttons:

Cancel - Use this to exit the dialogue without accepting any changes.

Factory settings - This will revert the dialogue settings to the initial default settings.

OK - This will close the dialogue and save all changes.

See also: Setting preferences

 

Check for Ambulant Update...

Using this feature will allow you to ensure that you have the latest version of the Player. Should there be a later version you will be taken to the relevant location from where you can perform a download of the new Player.
