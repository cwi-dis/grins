File Menu

Using the commands in this menu will enable opening and closing of documents for playback in the Player. Using the Preferences menu option you can personalise your Player options for playing all documents.

Open...

Using this command you can open files which reside on your hard drive or local network. Selecting a file with the extension .smi, .smil, or .grins will be opened in the player ready for playing.

See also: Opening files

 

Open URL...

This allows you to open a file which is on the internet. Using this dialog means you do not have to use your Internet Browser. You can also copy a URL from a other document and paste the location here which will then be loaded into the player ready for playing.

See also: Opening files

 

Open recent

Once files have been opened the last 5 of these files are listed in this sub menu, both local and remote file locations are listed here. Selecting a file from this menu will re-open the file ready for playing again.

See also: Opening files

 

Close document

This closes the currently selected document.

 

Preferences...

This dialog enables you to set the following options for playback of all documents:

System bitrate: To improve the quality of playback of a document choose one of the fololwing options which matches your connection type:

14.4K modem, 19.2K Connection, 28.8K Modem, 33.6K Modem, 56K Modem, 56K Single ISDN, 112K Dual ISDN, 256Kbps DSL/Cable, 300Kbps DSL/Cable, 512Kbps DSL/Cable, T1/LAN, 10Mbps LAN

System language: Choose your preferred language for receiving information. The default is set to 'English'.

Subtitles preferred over overdub :Turning this option on will allow you to receive overdub information which may be included in the document to be played.

Show closed-captions when available: Checking this option wil permit the viewing of captions which may be part of the document.

Use audio descriptions when available: Setting this option to on will enable you to view descriptions of audio which may be incorporated in the document.

Use Internet Explorer HTML control: Checking this option will change the control of HTML information to Internet Explorer, the difference will result in the displaying scroll bars on regions which contain html text.

Dialog buttons:

Cancel - Use this to exit the dialog without accepting any changes.

Factory settings - This will revert the dialog settings to the initial default settings.

OK - This will close the dialog and save all changes.

See also: Setting preferences

 

Check for GRiNS Update...

Using this feature will allow you to ensure that you have the latest version of the player. Should there be a later version you will be taken to the relivant location from where you can perform a download of the new player.
