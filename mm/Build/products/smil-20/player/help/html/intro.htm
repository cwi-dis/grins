Help Introduction

This help section will allow you to find information on the menu commands in the Player. There is a section on using the player which will help introduce the Preferences dialog which can be used to personalize your setup for playing back documents.

 

How to use this help

Selecting a topic heading on the left will expand out to give either another level of heading or a list of sub topics which when clicked on will display specific information in the window on the right.

 

System requirements

Windows

Minimum system requirements are a 120MHz Intel Pentium processor or better with 32MB or more of RAM and Windows 95, Windows 98, or Windows NT* 4.0 with Service Pack 3. Additionally the following are recommended: 28.8Kbps or better modem 16-bit or better sound card and speakers 65,000-color (High Color/16-bit) or better video display card Internet connection and web browser.

Macintosh

The minimum system requirements are a 200Mhz PPC Macintosh with 32 Mb of real memory, 65 Mb of virtual memory and MacOS 8.5. 64Mb of real memory and MacOS 8.5 are recommended, and a faster processor will result in better preview playback.
