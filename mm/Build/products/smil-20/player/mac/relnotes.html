GRiNS Player beta for SMIL 2.0, Release Notes for Macintosh

These are the release notes for GRiNS Player beta for SMIL 2.0, Release Notes for Macintosh, build 10. A more recent version of these release notes, as well as more recent builds of the player, may be available through http://www.oratrix.com/GRiNS/SMIL-2.0/.

Note that this is an early beta, and the playback engine has not been tested very thoroughly with large documents. You may experience instabilities.

SMIL 2.0 Compatibility

This player supports the SMIL 2.0 Language Profile, with the following notes:
  • Structure functionality
    • Structure Module
      • To be Supplied
  • Metainformation functionality
    • Metainformation Module
      • To be Supplied
  • Timing functionality
    • BasicInlineTiming Module
      • To be Supplied
    • SyncbaseTiming Module
      • To be Supplied
    • EventTiming Module
      • To be Supplied
    • MinMaxTiming Module
      • To be Supplied
    • WallclockTiming Module
      • To be Supplied
    • MultiArcTiming Module
      • To be Supplied
    • BasicTimeContainers Module
      • To be Supplied
    • ExclTimeContainers Module
      • To be Supplied
    • PrevTiming Module
      • To be Supplied
    • RestartTiming Module
      • To be Supplied
    • SyncBehavior Module
      • To be Supplied
    • SyncBehaviorDefault Module
      • To be Supplied
    • RestartDefault Module
      • To be Supplied
    • FillDefault Module
      • To be Supplied
  • Animation functionality
    • BasicAnimation Module
      • The target attribute soundLevel is recognized but not implemented.
      • The attribute attributeType is recognized but ignored (attributeType is always XML).
  • Content Control functionality
    • BasicContentControl Module
      • To be Supplied
    • CustomTestAttributes Module
      • To be Supplied
    • PrefetchControl Module
      • To be Supplied
    • SkipContentControl Module
      • To be Supplied
  • Layout functionality
    • BasicLayout Module
      • The scroll value for fit attribute is recognized but not implemented.
      • The z order may be wrong for several sibling regions with the same z-index value
    • AudioLayout Module
      • The soundLevel attribute is recognized but ignored.
    • MultiWindowLayout Module
      • Fully supported
    • HierarchicalLayout Module
      • To the media with no intrinsic sizes (html, text, ...), the registration point attributes are recognized but ignored
      • To the real medias, the registration point attributes work only if the fit value is equal to hidden (default value)
  • Linking functionality
    • BasicLinking Module
      • To be Supplied
    • LinkingAttributes Module
      • To be Supplied
    • ObjectLinking Module
      • To be Supplied
  • Transition Effects functionality
    • BasicTransitions Module
      • Subtypes are recognized but not implemented, the default subtype for the transition type is always used.
      • Type miscShapeWipe is recognized but not implemented.
      • Timing for most of the iris wipes is incorrect.
      • The direction attribute is recognized but ignored (direction is always forward).
      • Specifying multiple transitions in the transIn and transOut attributes is not supported.
      • When both the from and the to source of a transition are videos the from source will usually be frozen.
    • CoordinatedTransitions Module
      • Fully supported
    • TransitionModifiers Module
      • The TransitionModifiers attributes are recognized but ignored.
  • Media functionality
    • BasicMedia Module
      • To be Supplied
    • MediaClipping Module
      • To be Supplied
    • MediaParam Module
      • To be Supplied
    • BrushMedia Module
      • To be Supplied
    • MediaAccessibility Module
      • To be Supplied
