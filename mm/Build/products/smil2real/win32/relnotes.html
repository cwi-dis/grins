GRiNS Editor for RealONE, Release Notes for Windows

These are the release notes for GRiNS Editor for RealONE, Release Notes for Windows, build 98, September 24, 2001. A more recent version of these release notes, as well as more recent builds of the editor, may be available through http://www.oratrix.com/.

IMPORTANT NOTE: As of the release of this version of GRiNS/RealONE, the RealONE player architecture is in an "evolving" state. As a result, the preview experience in the GRiNS editor may be different from what you see in RealONE. We expect that these difference will become minimal by the time RealONE is in final product release.

SMIL 2.0 Compatibility

This player supports the SMIL 2.0 Language Profile, with the following notes:
  • Structure functionality
    • Structure Module
      • Fully supported
  • Metainformation functionality
    • Metainformation Module
      • Fully supported
  • Timing functionality
    • BasicInlineTiming Module
      • Fully supported
    • SyncbaseTiming Module
      • Fully supported
    • EventTiming Module
      • Fully supported
    • MinMaxTiming Module
      • Fully supported
    • WallclockTiming Module
      • Fully supported
    • MultiArcTiming Module
      • Fully supported
    • BasicTimeContainers Module
      • Fully supported
    • ExclTimeContainers Module
      • Fully supported
    • PrevTiming Module
      • Fully supported
    • RestartTiming Module
      • Fully supported
    • SyncBehavior Module
      • Only syncBehavior="independent" is supported
    • SyncBehaviorDefault Module
      • Fully supported
    • RestartDefault Module
      • Fully supported
    • FillDefault Module
      • Fully supported
  • Animation functionality
    • BasicAnimation Module
      • The attribute attributeType is recognized but ignored (attributeType is always XML).
  • Content Control functionality
    • BasicContentControl Module
      • Fully supported
    • CustomTestAttributes Module
      • Fully supported
    • PrefetchControl Module
      • Recognized
    • SkipContentControl Module
      • Fully supported
  • Layout functionality
    • BasicLayout Module
      • Fully supported
    • AudioLayout Module
      • Fully supported
    • MultiWindowLayout Module
      • Fully supported
    • HierarchicalLayout Module
      • For media with no intrinsic sizes (html, text, ...), the registration point attributes are recognized but ignored
  • Linking functionality
    • BasicLinking Module
      • Fully supported
    • LinkingAttributes Module
      • Fully supported
    • ObjectLinking Module
      • Fully supported, but only links in HTML documents can be overridden using the fragment attribute.
  • Transition Effects functionality
    • BasicTransitions Module
      • Subtypes are recognized but only some are implemented, the default subtype for the transition type is used in all other cases.
      • Type miscShapeWipe is recognized but not implemented.
      • Timing for most of the iris wipes is incorrect.
      • The direction attribute is recognized but ignored (direction is always forward).
      • Specifying multiple transitions in the transIn and transOut attributes is not supported.
      • When both the from and the to source of a transition are videos the from source will usually be frozen.
    • TransitionModifiers Module
      • The TransitionModifiers attributes are recognized but ignored.
  • Media functionality
    • BasicMedia Module
      • Fully supported
    • MediaClipping Module
      • Fully supported
    • MediaParam Module
      • Fully supported
    • BrushMedia Module
      • Fully supported
    • MediaAccessibility Module
      • Fully supported in that the readIndex attribute is parsed correctly. There is no support for actually reading out the alt, title, and longdesc attributes.
