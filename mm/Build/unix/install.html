GRiNS Editor and Player for SMIL Version 1.0 Installation Instructions

First, download the appropriate distribution. The following files are available:

grins-sgi-full-10.tardist
GRiNS Editor and Player for SGI, inst package
grins-sgi-full-10.tgz
GRiNS Editor and Player for SGI, gzipped tar
grins-sun-full-10.tgz
GRiNS Editor and Player for Sun, gzipped tar
grins-sgi-play-10.tardist
GRiNS Player for SGI, inst package
grins-sgi-play-10.tgz
GRiNS Player for SGI, gzipped tar
grins-sun-play-10.tgz
GRiNS Player for Sun, gzipped tar

The SGI version was created and tested on an SGI O2 running IRIX 6.5.2, the Sun version was created and tested on a Sun running Solaris 2.5 (SunOS 5.5).

For IRIX-6.5 (and later) on SGI systems, you have two options when installing the GRiNS Editor and Player: using the SGI inst tools or using the gzipped tar distribution. For Sun/Solaris the only option is usgin a gzipped tar distribution.

Using the SGI Inst Tools

Download the appropriate tardist file. If your Web browser is set up correctly, this will start the SGI Software Manager automatically. If not, you can extract the tardist file (it is just a tar file) in a new directory and start the Software Manager (swmgr) or inst manually. Point either program to the directory where the extracted files reside to start the installation process.

From the Software Manager or inst, select the subsystems you want to install (in the player-pnly version, the product is called grins instead of grinsed and the editor subsystem is not present):

grinsed.sw.player
contains the GRiNS Player and is required
grinsed.sw.editor
contains the GRiNS Editor and is required
grinsed.sw.templates
contains editing templates for the GRiNS Editor and is required
grinsed.help
contains help information and is recommended
grinsed.relnotes
contains the release notes and is also recommended

Important Note: You need superuser (root) privileges to use the inst installation method. The GRiNS Player and Editor (and the supporting files) will be installed in /usr/local/grins.

Using the Gzipped Tar File

Download the appropriate file and extract. The command to extract the gzipped tar file is something like this: 

gzip -cd grins*.tgz | tar xf -

The files will be extracted into a subdirectory called grins-1.0. The gzipped tar file contains the same files as the corresponding tardist file.
