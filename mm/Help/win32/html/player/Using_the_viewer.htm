When the view is launched the user must select open file to view the file to played. (the file can be double clocked to open the player automatically.

The user can browse to the file they want to view, or type the name in directly.

The player window will open to the correct size to play the document.

The icons allow the user to Open a new document, Close this document, Play, Pause or Stop

The menu items are as follows:

  • File menu:
    • Open - Open an existing document. The already opened session of the editor will not be closed, this allows more than one document can be edited at the same time.
    • Close - Quit the document. If the document has been changed a dialog box appears asking the user to save the changes.
    • Preferences - select the internet preferences for the system
    • Debug:
      • Trace - Trace option needed only by developers.
      • Debug - Debugging option needed only by developers.
      • Open debug window - show debug window messages.
    • Exit - Quit the editor.
  • Play menu:
    • Pause - Pause the playing document
    • Stop - Stop the playing document
  • Views menu: list of windows that can be viewed
  • Help menu:
    • About - info about system
