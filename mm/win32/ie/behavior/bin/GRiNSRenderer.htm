Click and drag any of the four black handles below.

A rendering behavior is attached to this paragraph element. The behavior initially creates a rectangular polygon that hides the text in this paragraph. The polygon has round handles at each corner that can be dragged to reshape the polygon and reveal the paragraph contents. Note that the handles' center points are located at the corners of the element; the handles actually extend outside the element's boundaries into an extended drawing region. For this reason, event handling for this element must rely on the IHTMLPainter::HitTestPoint method to test for mouse clicks on the handles in this extended region. Note also that while you drag a handle, the mouse cursor's movement is restricted to the extended region of the element using the Win32 API function ClipCursor.
